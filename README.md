# TabNews Case Study

A case study project based on TabNews.

## Prerequisites

This project requires **Node.js** to run.

* **Required Node Version**: `>=18.14.0` (LTS/Hydrogen v18.14.0 is recommended and specified in the `.nvmrc` file).

### Setting up Node.js with nvm

If you use [nvm](https://github.com/nvm-sh/nvm), you can automatically switch to the correct Node.js version by running:

```bash
nvm use
```

If you don't have the version installed, run:

```bash
nvm install
```

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) v13.1.6
- **Library**: [React](https://reactjs.org/) v18.2.0
